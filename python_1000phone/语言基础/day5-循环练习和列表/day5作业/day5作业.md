5.已知一个数字列表，将所有元素乘二。

例如:names = ['张三', '李四', '大黄', '张三'] -> names = ['张三', '李四', '大黄']

例如: list1 = [97, 98, 99]  -> list1 = ['a', 'b', 'c']

例如: A = [1, 'a', 4, 90]  B = ['a', 8, 'j', 1] --> C = [1, 'a']

例如: nums = [19, 89, 90, 600, 1]   —>  600

例如：nums = [1, 2, 3,1,4,2,1,3,7,3,3]   —> 打印:3
