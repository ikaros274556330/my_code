# Linux定时任务

通过制定 ```crontab -e``编辑和开启定时任务

```shell
# 创建一个完整的定时任务，需要确定时间和任务
# 1.时间
 .---------------- minute (0 - 59)   # 分
 |  .------------- hour (0 - 23)     # 时
 |  |  .---------- day of month (1 - 31)  # 日
 |  |  |  .------- month (1 - 12) OR jan,feb,mar,apr ...  # 月
 |  |  |  |  .---- day of week (0 - 6) (Sunday=0 or 7) OR sun,mon,tue,wed,thu,fri,sat
 |  |  |  |  |
 *  *  *  *  * user-name  command to be executed
 
 * * * * *   		- 每分钟执行一次任务(每隔一分钟)
 */1 * * * *        - 每分钟执行一次任务(每隔一分钟)
 */2 * * * *        - 每2分钟执行一次任务(每隔2分钟)
 * */1 * * *        - 每隔1小时
 * * */1 * *        - 每隔1天
 * * * */2 *        - 每隔两个月
 * * * *  fri       - 每周五执行任务
 
 30 * * * *         - 时间点到30分的时候做
 0  * * * *         - 时间点到整点的时候执行任务
 30 2 * * *         - 时间到 02:30 的时候执行任务
 0  12 28 * *       - 每月28号12:00执行任务
 
 
# 2.任务
写Linux指令, 指令如果有结果，结果看不到

```
